# AD-TabFusion 项目简介

## 一句话概述

AD-TabFusion 使用纵向多模态临床表格数据，预测未来 CN/MCI/AD 诊断状态与 MCI 向 AD 的进展风险，并通过严格时间验证、校准和选择性预测呈现模型可信度。

## 为什么做

阿尔茨海默病相关队列通常同时包含人口学、认知量表、MRI 派生指标和遗传信息，但记录时间不齐、模态缺失明显，而且同一受试者会反复随访。项目重点不是单纯追求随机划分下的最高分，而是建立不跨受试者、不读取未来信息、能在时间外推条件下审计的建模流程。

## 数据与任务

- 主要活跃模态：demographic、cognitive/clinical、MRI-derived、APOE4。
- 稀疏模态：PET、CSF、DTI，仅用于受限队列分析，不作为完整主模态宣称。
- 转归任务：基于当前状态和 6–60 个月预测跨度，预测未来 CN/MCI/AD。
- MCI 风险任务：评估 12、24、36、48 个月内从 MCI 进展至 AD 的风险。

## 核心结果

| 结果 | 证据划分 | 数值 |
|---|---|---:|
| 转归模型 Macro F1 | 锁定时间测试 | 0.881 |
| 转归模型 ROC-AUC OvR | 锁定时间测试 | 0.958 |
| 24 个月 MCI 风险 ROC-AUC | 锁定时间测试，n=53 | 0.797 |
| 36 个月 MCI 风险 ROC-AUC | 锁定时间测试，n=44 | 0.804 |
| 等渗校准 Log Loss | 锁定时间测试 | 0.352 |
| 80% 目标覆盖的实际覆盖 / Macro F1 | 锁定时间测试 | 81.1% / 0.926 |

## 科研边界

模型、校准器和选择性预测阈值只在验证集上确定；锁定时间测试集不参与再拟合。D4 的所有结果均为 **exploratory post-hoc replay**，不是独立验证性外部验证。项目是回顾性科研软件，不用于个体临床决策。

## 展示方式

Streamlit Dashboard 提供中英文切换，仅读取 `outputs` 中冻结 CSV/JSON。所有公开页面和截图只展示聚合结果，不出现 RID、PTID 或参与者级记录。
