# AD-TabFusion 四分钟演示脚本

## 0:00–0:45｜执行摘要

这是 AD-TabFusion，一个面向纵向临床表格数据的阿尔茨海默病转归研究项目。我们关注两个问题：未来 6 到 60 个月的 CN、MCI、AD 状态，以及 MCI 在不同时间窗内进展为 AD 的风险。首页只显示冻结结果：主要转归模型在锁定时间测试集上的 Macro F1 为 0.881，ROC-AUC OvR 为 0.958。Dashboard 不训练模型，只读取已有 artifact。

## 0:45–1:45｜转归感知模型

进入 Transition-Aware Model。这里首先看到锁定时间测试的五项核心指标和三类诊断的 precision、recall、F1。消融结果说明，单纯使用表格特征的 Macro F1 约为 0.645；加入当前诊断状态后达到 0.810，再加入预测时间跨度达到 0.820。这个结果支持将“当前状态”和“预测多远”明确编码进纵向转归任务。

## 1:45–2:35｜MCI 进展风险

MCI Progression Risk 页面按 12、24、36、48 个月展示。24 和 36 个月锁定测试 ROC-AUC 分别为 0.797 和 0.804。页面同时展示可评估人数、转化人数、未转化人数和删失人数。48 个月只有 22 名测试受试者，因此明确标记为小样本探索性结果，不能与较短窗口等量齐观。

## 2:35–3:25｜校准与不确定性

Calibration & Uncertainty 页面强调预测分数是否可信。等渗校准把锁定测试 Log Loss 从 0.373 改善到 0.352，ECE 从 0.047 改善到 0.039，但 Brier Score 基本不变。选择性预测的 80% 目标覆盖阈值只在验证集确定，应用到测试集后实际覆盖 81.1%，保留样本的 Macro F1 为 0.926，错误率为 7.3%。

## 3:25–4:00｜科研边界

最后看 Scientific Guardrails 和 External Replay。数据按受试者隔离，未来信息不进入特征，模型、校准器和阈值都不使用测试集选择。D4 始终标记为 exploratory post-hoc replay，不是独立的验证性外部验证。公开页面和截图只包含聚合结果，不显示 RID、PTID 或参与者级记录。
